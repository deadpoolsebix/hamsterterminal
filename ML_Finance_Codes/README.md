# ML Finance Codes - Kodowanie w Finansach

Kodami z książki "Machine Learning in Finance: From Theory to Practice"

## Zawartość

- `1_Data_Collection.py` - Pobieranie i przygotowanie danych
- `2_Feature_Engineering.py` - Inżynieria cech
- `3_Model_Training.py` - Trenowanie modeli ML
- `4_Backtesting.py` - Testowanie strategii
- `5_Trading_Bot.py` - Bot tradingowy

## Zależności

Wszystkie wymagane biblioteki są już zainstalowane w środowisku venv projektu.

```bash
# Aktywuj środowisko
.\venv\Scripts\Activate.ps1

# Uruchom przykłady
python 1_Data_Collection.py
```

## Struktura

```
ML_Finance_Codes/
├── data/                    # Dane do analizy
├── models/                  # Zapisane modele
├── results/                 # Wyniki backtestingu
└── *.py                     # Skrypty Python
```
